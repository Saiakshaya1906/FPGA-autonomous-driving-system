# 🚗 FPGA-Based Autonomous Driving System

## 📌 Project Description

This project implements a **Supervised Learning-Based Autonomous Driving System** with **real-time FPGA output**.

The system processes input images, predicts steering directions (Left / Straight / Right), and sends the output to an FPGA board for display.

---

## 🧠 System Workflow

1. Image Capture (PyGame Simulation)
2. Image Preprocessing (OpenCV)
3. ML Model Prediction
4. Serial Communication
5. FPGA Processing
6. Output Display (7-Segment)
